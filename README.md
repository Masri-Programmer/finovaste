Developer: # Occasion and Fundraising Types Overview

There will be several types of listings:

- **Private Occasions**
- **Charity Actions**
- **Donation Campaigns**
- **Founders & Creatives**

## Details and Purposes

All types serve the same core purpose: enabling fundraising and donations. Here‚Äôs an explanation of each type:

### 1. Private Occasion

- A private listing specific to a user and their family and friends.
- Only users with the unique link can access these listings (no authentication required).
- Example link: `ranalp.com/listings/xyzw-xfg`
- Used to collect donations from anyone who has the link.

### 2. Charity Action

- Listings for direct purchases or auction bids.
- Funds raised through charity actions can be transferred to a donation campaign.

### 3. Donation Campaign

- Can only be created by clubs.
- Clubs must be authenticated by ID and supporting documents (to be implemented later).
- Focused on formal fundraising by verified organizations.

### 4. Founders & Creatives

- Users share their story, vision, or creative project.
- Enables people to donate or fund their foundations, plans, projects, or ideas.
